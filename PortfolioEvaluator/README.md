# Portfolio Evaluator

## Project Overview
This project aims to explore the various metrics used in the evaluation of a portfolio.

It investigates and codes key metrics such as:
- Cumulative Return
- Average Daily Return
- Sharpe's Ratio
- Bollinger Bands
- Volatility
- Daily Portfolfio Value
